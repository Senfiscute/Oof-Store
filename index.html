What is Oof Store?

Oof Store is a game library.

Oof Store is a game library with a variety of games. We have 2d and 3d games. No matter your style of games, whether its endless games with highscores or games with a goal to complete, we got it! You won't get bored!

How did we make these games?

All of the games on the Oof Store were made using Unity. What's unity you ask? Unity is a game developing platform used to make all kind of games, many famous games were made on Unity. On Oof Store we have used Unity's 2D and 3D gamebuilding options to create a variety of fun games. We hope you enjoy!

How did we make this website?

This website was made using some of the two most simple coding languages that exist, HTML and CSS. HTML and CSS are coding languages that are used to make websites. Using the properties we learned of CSS and HTML we created Oof Store!
